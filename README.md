# avpkg
Available Packages

List available packages in the fpm-registry.

## Build
```bash
git clone https://github.com/brocolis/avpkg
cd avpkg
fpm build
```

## Usage
`$ avpkg`


## Dependencies
`
fortran-curl = { git = "https://github.com/interkosmos/fortran-curl" }
M_strings = { git = "https://github.com/urbanjost/M_strings.git" }
`
